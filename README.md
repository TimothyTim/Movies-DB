# Movies DB

## Setup

First add your MoviesDB API key to: `env.config.ts`.

Then run the following to install project dependencies:

```
npm install
```

## Running

To start the application:

```
npm start
```

Run the following to create a production bundle of the application:

```
npm run build
```
